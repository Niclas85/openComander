OpenCommander is a free and open-source file manager for Android. It is built for people who want to organize local files without an account, ads, tracking or forced cloud sync.

The dual-pane layout gives you a classic commander-style workflow: each side has its own folder tree and file list. In landscape mode both sides sit next to each other; in portrait mode they are stacked for smaller screens.

Key features:

- Dual-pane file management
- Folder tree and file list on both sides
- Copy or move mode
- Multi-select by tapping files
- Drag & drop between panes
- Delete with cancel, delete and trash choices
- Undo with action history
- Progress with percent, item count and bytes
- Browse ZIP archives like folders
- Create ZIP archives from selected files and folders
- File type icons for images, videos, music, PDF, Office files, archives and more
- Light and dark theme
- Multilingual interface
- No ads, no tracking, no account

OpenCommander works locally on your device. It does not upload files to provider servers and does not use analytics SDKs. File access is required only so the file manager can show, open, copy, move and archive files.
