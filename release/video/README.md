# OpenCommander Generated Media

Generated from real app screenshots because this repository currently has no local Gemini/Veo automation scripts.

Files:
- `opencommander-promo.mp4`: 24s portrait promo video
- `opencommander-dual-pane.mp4`: short dual-pane clip
- `opencommander-undo.mp4`: short undo/trash clip
- `opencommander-zip.mp4`: short ZIP workflow clip

For Google Play promo video, upload the chosen MP4 to YouTube and enter the YouTube URL in Play Console.
