# Contributing to OpenCommander

Thanks for helping improve OpenCommander.

## Good First Contributions

- Improve translations.
- Report bugs with clear reproduction steps.
- Test file operations on different Android versions.
- Improve app store screenshots or copy.
- Suggest small usability improvements.

## Bug Reports

Please include:

- Android version and device model.
- What you tried to do.
- What happened.
- What you expected.
- Whether the issue happened with files, folders or ZIP archives.

## Feature Requests

Please describe:

- The workflow you want to improve.
- Why it matters.
- Whether it should work in portrait, landscape or both.
- Any privacy or permission implications.

## Development

Build debug APK:

```powershell
.\gradlew.bat :app:assembleDebug --no-daemon
```

OpenCommander should remain:

- free
- open source
- local-first
- without ads
- without tracking
- usable without an account
