# Google Play All Files Access Declaration

## Permission

`MANAGE_EXTERNAL_STORAGE`

## App Category

File manager

## Short Justification

OpenCommander is a local dual-pane file manager. All files access is required for the app's core purpose: browsing folders, copying files, moving files, creating ZIP archives, opening ZIP archives and undoing file operations across shared storage locations selected by the user.

## German Explanation

OpenCommander ist ein lokaler Dateimanager. Der Zugriff auf alle Dateien ist erforderlich, damit Nutzer Ordner durchsuchen, Dateien kopieren, Dateien verschieben, ZIP-Archive erstellen, ZIP-Archive oeffnen und Dateiaktionen rueckgaengig machen koennen. Diese Funktionen sind die Kernfunktion der App.

Die App nutzt den Dateizugriff nur fuer vom Nutzer ausgeloeste lokale Dateioperationen. OpenCommander uebertraegt keine Dateien an Anbieter-Server, verwendet keine Werbung, kein Tracking und keine Analyse-SDKs.

## User-Facing Permission Explanation

OpenCommander benoetigt Dateizugriff, damit du lokale Dateien und Ordner anzeigen, kopieren, verschieben, zippen und wiederherstellen kannst. Die App arbeitet lokal auf deinem Geraet und uebertraegt keine Dateien an eigene Server.

## Review Notes
