## Berechtigungen

### All Files Access

- Berechtigung: `MANAGE_EXTERNAL_STORAGE`
- Kategorie: File manager
- Kernfunktion: Lokaler Zwei-Seiten-Dateimanager
- Deklarationstext:

OpenCommander is a local dual-pane file manager. All files access is required for the app's core purpose: browsing folders, copying files, moving files, creating ZIP archives, opening ZIP archives and undoing file operations across shared storage locations selected by the user. The app does not upload files to provider servers and does not use ads, tracking or analytics.

### Nutzererklaerung

OpenCommander benoetigt Dateizugriff, damit du lokale Dateien und Ordner anzeigen, kopieren, verschieben, zippen und wiederherstellen kannst. Die App arbeitet lokal auf deinem Geraet und uebertraegt keine Dateien an eigene Server.
