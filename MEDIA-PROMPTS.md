# OpenCommander Media Prompts

These prompts are prepared for Gemini image generation and Veo video generation. The current repository does not include local `gemini/` or `gemini-video/` automation scripts, so these prompts are ready to paste into Gemini manually or wire into future scripts.

## 1. Icon

Create a clean Android app icon for "OpenCommander", a free open-source dual-pane file manager. Show two minimal file panels side by side, a small folder symbol, and a subtle checkmark/undo cue. Use a professional flat style, high contrast, rounded Android icon composition, no text, no brand names, no third-party logos. Colors: fresh blue, teal, white, dark neutral accents. Output square, suitable for 512x512 Play Store icon.

## 2. Feature Graphic

Create a Google Play feature graphic for "OpenCommander", a free open-source dual-pane Android file manager. Show the actual product idea clearly: two file panels, folder trees, file rows, drag-and-drop movement, ZIP archive cue, and undo history. Include tasteful text: "OpenCommander" and "Dual-Pane File Manager". Small badges: "Open Source", "No Ads", "Local Files". Clean modern utility-app design, bright readable layout, not dark, no third-party logos, no phone brand logos. Output 1024x500.

## 3. Phone Screenshots

Create polished Google Play phone screenshot compositions for OpenCommander using real app UI as the central content. Each image should look like a real Android file manager screen with clear dual-pane workflow, compact buttons, folder tree, file list, ZIP, trash and undo. Add short top captions, large enough to read on the Play Store:

1. "Two panes. Faster file work."
2. "Copy or move files."
3. "Delete. Trash. Undo."
4. "Open ZIPs like folders."
5. "Free. Open source. No ads."

Use 1080x1920 portrait format, clean white or light neutral background, subtle blue/green accents, no fake reviews, no third-party trademarks.

## 4. Promo Video

Create a 25-30 second portrait promo video for OpenCommander, a free open-source dual-pane file manager for Android. Use real product-style UI scenes, not abstract animation.

Timing:
- 0-4s: App opens in dual-pane view. On-screen text: "Two panes. Faster file work."
- 4-9s: User selects files and drags them to the other pane. Text: "Copy or move files."
- 9-14s: Delete dialog appears with Cancel, Delete and Trash choices. Text: "Delete. Trash. Undo."
- 14-20s: Undo history restores a mistaken action. Text: "Undo file actions."
- 20-25s: ZIP archive opens like a folder. Text: "Open ZIPs like folders."
- 25-30s: Trust screen. Text: "Free. Open source. No ads."

Style: realistic Android app demo, readable UI, simple transitions, no music requirement, no third-party logos, no exaggerated claims.

## 5. Short Social Clips

### Dual Pane Copy

10-15 second portrait clip. Show OpenCommander in landscape or portrait with two panes. Select one file, drag it to the other pane, show progress. Text: "Two panes. Faster file work."

### Undo Mistakes

10-15 second portrait clip. Show a file action, open history, tap undo, restored file visible. Text: "Undo file actions."

### ZIP Workflow

10-15 second portrait clip. Show a ZIP file opening like a folder, then selected files being packed into a ZIP. Text: "Open ZIPs like folders."
